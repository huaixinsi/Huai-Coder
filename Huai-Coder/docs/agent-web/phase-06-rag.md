# 阶段六：代码 RAG

## 目标

让 Agent 能够对项目代码建立索引，并通过自然语言完成跨文件检索。

## 技术方案

- 代码解析：Tree-sitter、Python AST。
- 结构化索引：PostgreSQL。
- 关键词检索：PostgreSQL Full-Text Search。
- 向量检索：后续使用 pgvector。
- 检索策略：路径、关键词、结构和向量混合检索。

## 后端任务

1. 实现文件扫描。
2. 实现代码分块。
3. 保存文档、代码块、语言和位置信息。
4. 实现索引任务和进度事件。
5. 实现关键词检索。
6. 后续接入 Embedding 和 pgvector。
7. 将检索结果适配为 Agent Tool。

## 前端任务

1. 实现索引启动和取消。
2. 展示索引进度。
3. 展示索引错误。
4. 展示检索命中的文件和代码位置。

## 验收

- 支持 Python、Java、JavaScript 基础代码索引。
- Agent 能检索跨文件代码。
- 索引过程可观察、可取消、可重试。
- 不同项目的数据互相隔离。
